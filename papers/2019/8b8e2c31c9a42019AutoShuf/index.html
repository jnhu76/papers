摘要

ShuffleNet是一种先进的轻量级卷积神经网络架构，其基本操作包括分组卷积、逐通道卷积和通道重排。然而，通道重排是经验性手动设计的。从数学上看，重排是乘以一个置换矩阵。本文提出通过在网络训练中学习置换矩阵来自动化通道重排。

核心创新点
  • 引入了精确的Lipschitz连续非凸惩罚函数，可整合到SGD中高精度近似置换矩阵
  • 训练结束后通过简单舍入获得精确置换矩阵，用于推理
  • 所得网络（称为AutoShuffleNet）在CIFAR-10和ImageNet数据集上实现了更高的分类精度

论文作者通过实验发现，将置换矩阵凸松弛为随机矩阵会导致性能不佳。作者从理论上证明了当惩罚函数为零（或非常小）时恢复置换矩阵的精确性（误差界限）。作者还通过图匹配和两层神经网络模型的示例，展示了凸松弛无法捕获置换矩阵，而作者的惩罚函数能够成功。

置换优化问题背景

置换优化是运筹学、图匹配等领域中长期存在的问题。经典的图匹配问题可表述为：给定两个图的邻接矩阵A和B，寻找置换矩阵π最小化‖A-πBπᵀ‖²_F。处理置换矩阵的常用方法是将其松弛为双随机矩阵（Birkhoff多面体），但这会导致凸松弛，且由于目标函数的非凸性，问题仍然复杂。

问题与挑战

技术挑战1：离散性与连续优化的冲突

  • 离散性：置换矩阵是高度离散的数学对象（只有0和1）
  • 连续性要求：深度学习训练基于随机梯度下降(SGD)，本质上是连续的
  • 不可微性：置换操作不可微，无法直接通过反向传播优化

技术挑战2：现有松弛方法的局限性

  • 凸松弛失效：将置换矩阵松弛为双随机矩阵（凸包）在LCNN中效果不佳
  • 计算复杂度：Sinkhorn等矩阵缩放算法迭代成本高，造成瓶颈效应
  • 非精确性：现有松弛方法都不是精确的，无法保证恢复真正的置换矩阵

置换矩阵的数学特性

特性 数学描述 约束条件
置换矩阵P P ∈ {0,1}N×N 每行每列恰好有一个1
双随机矩阵D D ∈ [0,1]N×N 行和=1，列和=1，非负
关系 P ⊂ D 置换矩阵是双随机矩阵的子集

关键洞察：双随机矩阵是置换矩阵的凸包，但单纯使用双随机约束无法在优化过程中引导矩阵收敛到置换矩阵（顶点）。需要额外的正则化来"推动"双随机矩阵向顶点（置换矩阵）移动。

设计与实现

ℓ₁-₂惩罚函数：核心数学创新

数学定义

矩阵 ℓ₁-₂ 惩罚函数：

P(M) := Σi=1N [Σj=1N|mij| - (Σj=1Nmij2)1/2] +
Σj=1N [Σi=1N|mij| - (Σi=1Nmij2)1/2]

双随机约束：

mij ≥ 0, ∀(i,j) (非负性)
Σi=1N mij = 1, ∀j (列和为1)
Σj=1N mij = 1, ∀i (行和为1)

关键定理与证明

定理1：精确性条件

方阵M是置换矩阵当且仅当P(M)=0且双随机约束成立。

证明思路

⇒ 方向：置换矩阵的每行每列恰好有一个1，其余为0，直接计算可得P(M)=0。

⇐ 方向：由Cauchy-Schwarz不等式，对于行i：

Σj|mij| - (Σjmij2)1/2 ≥ 0

等号成立当且仅当行基数=1，即每行只有一个非零元素。结合双随机约束，该非零元素必须为1。同理应用于列，可得M是置换矩阵。

定理2：误差界限

如果P(M)=O(ε)，ε≪1，且双随机约束满足到O(ε)，则存在置换矩阵P*使得‖M-P*‖=O(ε)。

该定理保证了当惩罚函数值很小时，所得矩阵非常接近某个置换矩阵，为训练结束后的舍入操作提供了理论保证。

AutoShuffle学习算法

算法流程

算法1: AutoShuffle Learning
伪代码
1
输入：小批量损失函数f_t(w,M)，学习率η^t，惩罚参数λ，总迭代数Tn
2
初始化：
  • w：从单位高斯分布采样
  • M：从单位高斯分布采样后取绝对值
3
循环训练（t < Tn）：
  1. 计算梯度(∇_w f_t, ∇_M f_t)
  2. 更新权重：w^{t+1} = w^t - η^t ∇_w f_t
  3. 更新M：M^{t+1} = M^t - η^t ∇_M f_t
  4. 阈值化：M^{t+1} ← max(M^{t+1}, 0)
  5. 列归一化：每列除以其和
  6. 行归一化：每行除以其和
4
输出：训练后的w^{Tn}和M^{Tn}，将每个M_j^{Tn}投影到最近的置换矩阵

算法关键特性

交替归一化

采用Sinkhorn过程（RAS方法）的简化版本：每次迭代只进行一次列归一化和一次行归一化。虽然理论上需要多次迭代收敛，但实验发现单次迭代在SGD背景下已足够。

惩罚参数选择

λ选择为10⁻³或2×10⁻³，以平衡损失函数和惩罚项的贡献，并驱动ΣP(M_j)接近0。λ太小会导致惩罚不足，太大可能影响主要任务的学习。

网络集成

将M的乘法嵌入网络作为1×1卷积层。初始化时，M取随机高斯矩阵的绝对值。在训练中，M与网络权重一起通过SGD优化。

技术解释：
  • 函数定义: F(q) = 6q² - 8q + 2 - 4λ(q² + (1-q)²)^{1/2}，其中q∈[0,1]表示双随机矩阵参数
  • λ=0.25: 最小点在区间内部(q≈0.67)，对应凸松弛解，不是置换矩阵
  • λ=1: 最小点移动到端点q=1，对应置换矩阵解
  • λ=2: 最小点保持在q=1，稳定地恢复置换矩阵
  • 关键发现: 当λ足够大时，ℓ₁-₂惩罚能够引导优化收敛到置换矩阵，而凸松弛(λ=0)则无法做到

结论：在此示例中，凸松弛得到的解(q=2/3)不是置换矩阵。而使用ℓ₁-₂惩罚后，当λ≥1时，优化收敛到正确的置换矩阵解(q=1)。
